# Book Social Network

## 项目简介

Book Social Network 是一个基于 Spring Boot 的社交网络应用，专门为读书爱好者设计。用户可以在平台上分享图书、撰写评论，并发现新的阅读内容。

## 主要功能

- **用户管理**：用户注册、登录、个人资料管理
- **图书管理**：添加、编辑、删除图书信息
- **社交功能**：关注/取消关注其他用户
- **评论系统**：对图书进行评论和评分
- **搜索功能**：搜索图书和用户

## 技术栈
### backend
- **后端框架**：Spring Boot 3
- **数据库**：PostgreSQL
- **安全框架**：Spring Security 6
- **ORM**：Spring Data JPA
- JSR-303 和 Spring Validation
- OpenAPI 和 Swagger UI 文档
- Docker
- Keycloak
